Kaliaff PWA install guide:
1) Upload ALL files in this folder together: index.html, manifest.json, sw.js, icon-192.png, icon-512.png, apple-touch-icon.png, kaliaff.jpg
2) Use HTTPS hosting: Netlify, Vercel, or Firebase Hosting.
3) Open the website in Android Chrome.
4) Tap the Download button inside the website OR Chrome three-dot menu > Install app / Add to Home screen.
5) If install option does not show, clear Chrome cache for the site and reload twice.

Important: PWA cannot be installed from file://. It must be on HTTPS or localhost.
